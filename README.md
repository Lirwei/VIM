# .vim

#### Basic Srttings
| Keys        | Descriptions                 |
| ----------- | -----------                  |
| sk          | 在上面分出工作区             |
| sj          | 在下面分出工作区             |
| sh          | 在左面分出工作区             |
| sl          | 在右面分出工作区             |
| \<C-h>       | 切换到左边的工作区           |
| \<C-l>       | 切换到右边到工作区           |
| \<C-j>       | 切换到下面的工作区           |
| \<C-k>       | 切换到上面的工作区           |
| \<leader>tm | 启用markdown表格自动适配功能 |
| \<leader>mp | 浏览器预览md文件             |
